## Prerequisites

- Java 17 or higher
- Maven 3.6 or higher

## Getting Started

1. Clone the repository.
2. Navigate to the demo directory.
3. Run `mvn spring-boot:run` to start the application.

The application will start on port 8080.

## API Endpoints

- GET /api/foyer - Get all foyer data
- POST /api/foyer - Create new foyer entry
